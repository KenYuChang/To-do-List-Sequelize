# To-do-List-Sequelize

## Introduction
Node.js, Express, handlebars, mySQL, Sequelize, CRUD, passport-local, passport-facebook, bcryptjs

## Features

1. 新增記帳功能
2. 編輯記帳功能
3. 刪除記帳功能
4. 新增註冊、登入功能

- Framework: express @ 4.16.4

- Database: mysql2 @2.1.0
- Check package.json for other dependencies
